Basic Usage:

```jsx
import TimeForm from "./"
import "src/styles/base/_fonts.scss"

milliseconds = 666666
;<TimeForm milliseconds={milliseconds}>basic button</TimeForm>
```

Small Usage

```jsx
import TimeForm from "./"
import "src/styles/base/_fonts.scss"

milliseconds = 666666
;<TimeForm milliseconds={milliseconds} size="small">
  basic button
</TimeForm>
```
